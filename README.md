## 🛠️ วิธีการติดตั้ง

### 1️⃣ ติดตั้ง VS Code
ดาวน์โหลดจาก https://code.visualstudio.com/

### 2️⃣ ติดตั้ง Dart SDK

**Windows (PowerShell):**
```powershell
choco install dart-sdk
```

**macOS (Homebrew):**
```bash
brew tap dart-lang/dart
brew install dart
```

**Linux (apt):**
```bash
sudo apt-get update
sudo apt-get install dart
```
หากใช้ choco ไม่ได้แนะนำให้ใช้ scoop หรือดาวน์โหลดจากเว็บไซต์ Dart โดยตรง

### 3️⃣ ติดตั้ง Extensions ใน VS Code
- **Dart** (Official Extension)
- **Code Runner** (สำหรับรันโค้ด)
- **Better Comments** (สำหรับ Comments)

### 4️⃣ ตรวจสอบการติดตั้ง
```bash
dart --version
```

ถ้าได้ผลลัพธ์เวอร์ชัน ✅ ติดตั้งสำเร็จแล้ว!

## 🚀 รันโปรแกรม Dart

```bash
dart filename.dart
```

หรือใน VS Code: **Ctrl + Alt + N** (Windows/Linux) / **Cmd + Ctrl + N** (macOS)
